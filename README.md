# PLE-Bloom

A gamified study habit app built with React Native (Expo). Bloom helps students build daily study routines through a cozy game-world experience, complete tasks, reflect on your progress, and watch your world grow.

## 🎮 Features

- **Daily task flow** — task input → completion → reflection, all in one seamless flow
- **XP system** — earn up to 100 XP per day based on completion status, time invested, reflection quality, and streaks
- **Level system** — 10 levels with increasing XP thresholds
- **Streak tracking** — consecutive daily streaks reward bonus XP
- **Visual world progression** — world evolves at each level milestone with new assets
- **Persistent storage** — daily progress and session history saved with AsyncStorage
- **Daily reset** — new day, new task; previous sessions are preserved
